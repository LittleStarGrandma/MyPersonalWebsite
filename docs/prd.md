﻿﻿﻿﻿﻿﻿﻿# 个人网站 PRD（V1）

## 1. 背景与目标
- **背景**：搭建一个可公开访问的个人网站，用于展示个人信息、项目经历、相册与社交主页链接。
- **目标**：
  - 让访客在 1 分钟内了解“你是谁、做过什么、在哪里能找到你”。
  - 信息可持续维护（数据与展示分离，后续好扩展）。
  - 支持多语言：简体中文（zh-CN）、繁体中文（zh-TW）、英文（en）。
- **非目标（本版本不做）**：
  - 账号体系/登录、后台 CMS、评论系统、复杂服务端。
  - 付费/电商、复杂的动态权限控制。

## 2. 受众与场景
- **受众**：面试官/猎头、潜在合作方、同行开发者、朋友。
- **核心场景**：
  - 快速浏览个人简介与技能栈
  - 查看代表项目（筛选、点击查看详情/外链）
  - 浏览相册（网格 + 放大预览）
  - 一键跳转到社交主页（GitHub / LinkedIn / X / Bilibili / 小红书等）

## 3. 信息架构（IA）
### 3.1 页面结构（单页滚动优先）
- **顶部导航**：个人信息 / 项目 / 相册 / 社交 + 语言切换
- **Hero/个人信息**
  - 姓名/昵称、头衔、城市（可选）、一句话介绍
  - 技能标签（可选）
  - CTA：下载简历 / 联系方式 / 邮箱复制
- **项目**
  - 项目卡片列表：标题、时间、简介、技术栈、角色、外链
  - 筛选：按标签（例如 Web / Mobile / AI / Tooling）
- **相册**
  - 照片网格：缩略图
  - 预览弹窗：左右切换、键盘 Esc 关闭
- **社交**
  - 社交链接列表/图标：GitHub、LinkedIn、X、Blog、Email 等
- **页脚**
  - 版权信息、备案（可选）、站点生成时间（可选）

### 3.2 数据模型（前端可维护）
- `profile`：姓名、头衔、简介、技能、联系方式
- `projects[]`：id、name、summary、period、tags、stack、links、highlights
- `album[]`：id、src、alt、takenAt（可选）、location（可选）
- `socials[]`：platform、label、url
- `i18n`：三语文案字典（按 key 管理）

## 4. 关键功能需求（FR）
### FR-1 多语言（必须）
- 支持 zh-CN / zh-TW / en。
- 语言切换入口：
  - 顶部导航右侧语言切换（下拉或分段按钮）
- 语言持久化：
  - 本地存储记忆上次选择
  - 首次访问默认：浏览器语言匹配，否则 zh-CN
- 可选增强（后续）：
  - URL 参数（`?lang=en`）或路径前缀（`/en`）以利分享

### FR-2 个人信息展示（必须）
- 基础信息：姓名/昵称、头衔、简介
- 联系方式：至少 1 个（邮箱/微信/telegram 等）
- 支持一键复制邮箱

### FR-3 项目展示（必须）
- 项目列表以卡片形式展示
- 支持标签筛选（多选或单选，V1 可做单选）
- 每个项目支持：
  - 访问链接（GitHub / Demo / 文章）
  - 亮点 bullet（2–4 条）

### FR-4 相册（必须）
- 网格展示
- 点击缩略图打开预览弹窗
- 弹窗支持：
  - 上一张/下一张
  - Esc 关闭
  - 点击遮罩关闭

### FR-5 社交主页（必须）
- 社交链接列表，点击打开新标签页

## 5. 交互与体验（UX）
- 导航点击平滑滚动到对应 section，并高亮当前 section（V1 可简化为点击滚动）
- 项目筛选即时生效（无刷新）
- 相册弹窗动画（V1 可用淡入淡出）
- 响应式：
  - 移动端优先，导航在窄屏折叠为菜单

## 6. 视觉与内容规范（UI）
- 风格：简洁、偏技术感、留白足
- 字体：系统字体优先
- 色彩：浅色主题优先；可选暗色主题（后续）
- 图片：统一长宽比缩略图（如 4:3 / 1:1）

## 7. 技术方案（前端）
- **形态**：静态站点优先（便于部署到 GitHub Pages/Vercel/Netlify）
- **数据**：本地 JSON/TS 常量，后续可接 CMS
- **i18n**：本地字典 + React Context（V1 不引入重型依赖）
- **SEO**（基础）：
  - title/description
  - Open Graph（后续）

## 8. 可配置项（V1）
- 站点标题、头像、简介
- 项目列表与标签
- 相册图片列表
- 社交链接

## 9. 里程碑（V1 交付物）
- PRD 文档
- 前端可运行原型：
  - 4 个 section + 多语言切换
  - 项目筛选 + 相册预览
- 基础构建与本地启动通过

## 10. 后续可扩展（你补充“有趣交互”时的落点建议）
- 动态背景（粒子/视差/渐变跟随鼠标）
- 项目卡片 3D 倾斜/hover 预览
- 相册时间轴/地图视图
- 终端风格介绍区（可切换主题）
- “彩蛋”交互：键盘 Konami code、点击头像触发动画等
